# 上下文记忆 Skill 导入记录

本项目把 GitHub 上的上下文记忆思路导入到 Anna 个人助理模式的强化学习项目中，落地为 `learning_status` / `learning_cycle`。

## GitHub 参考

- `SpillwaveSolutions/project-memory`：采用结构化项目记忆思路，把进度、决策、经验和后续规则保存为可读、可恢复的记录。
- `muratcankoylan/Agent-Skills-for-Context-Engineering/skills/memory-systems`：采用分层记忆思路，把事实、经验、规则和复盘摘要分开，读取时只取当前任务相关内容。
- `zilliztech/memsearch`：采用可重建记忆记录思路，让后续上下文增强有明确来源和进度记录。

## 导入到 Anna 的设计

1. 用户明确触发“强化学习 / 自主学习 / 学习复盘 / 训练”后，Anna 才执行学习循环。
2. 每轮学习读取书目级课程矩阵：心理学 5 本、逻辑学 5 本、回复能力 5 本。
3. Anna 根据学习原则对个人助理模式本身进行自测。
4. Anna 生成复盘，总结本轮经验和需要修正的回复规则。
5. 学习进度、复盘和强化规则写入本地 JSON 记忆，供后续上下文读取。
6. 普通回复只应用已有强化记忆，不冒充新一轮学习。

## 边界

- 不复制受版权保护的书籍正文。
- 不保存用户原始私人对话。
- 不上传私人语料作为训练数据。
- 不声称修改了 Anna 主机底层模型权重。
- 不绕过健康、隐私、付款、外站接管和用户确认边界。
